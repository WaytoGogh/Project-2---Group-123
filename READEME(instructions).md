1. Run the program.
2. Type in your specified genre exactly (no prefixes).

Examples of valid genres are "rock", "acoustic", "afrobeat", etc.

3. Press enter.

The list of songs from the specified genre should be printed. They will be printed in order
of most popular to least popular based on each song's popularity number (a larger number means the song is more popular). 
The performance comparision of how long it takes for each data structure to search for the genre is also printed 
below the list of songs. 

If an invalid genre name is typed in, a message will appear saying "no songs were found."

4. Type in another genre for its corresponding list of songs or type in "exit" if you are done.
